### 1.  Project 소개

'너의 문자는' 프로젝트는 스미싱 문자 사기 예방을 위해 open AI를 활용한 문자 분석 서비스입니다. 

사용자가 의심되는 문자를 입력하면, AI가 해당 문자의 사기 위험도를 분석하고 결과를 이미지와 테이블을 통해 시각적으로 보여줍니다. 

이 서비스는 사용자들이 스미싱 문자를 사전에 차단하고 안전하게 문자 서비스를 이용할 수 있도록 돕는 것을 핵심 목표로 합니다.
